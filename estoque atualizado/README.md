# 📦 API REST - Estoque
> Módulo de gerenciamento de estoque
> desenvolvido para a diciplina de **Sistemas distribuidos**, 
> responsável pelo controle de produtos em locais fisicos e pelo registro de entrada e saída de produtos.
> 
---
# 📄 Descrição 
O modulo Estoque gerencia três recursos:
- **Locais fisicos** - galpões
- **Estoque** - Quantidade de produtos 
- **movimentações** - Entrada e saída
